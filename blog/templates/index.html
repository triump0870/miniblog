KickStarter - Miniblog

A simple Personal Blog

New

This is my Personal Blog Project which is developed in Django1.7. The website is hosted in an AWS Linux instance under Nginx server. I used Virtualenv to make a isolated Django environment. Here I used Gunicorn to wrap the Django environment as Django is single threaded so it is incapable of handling multiple simultaneous requests. But Gunicorn server cant handle the static files that's why I used Nginx to serve the static files. Again the Gunicorn server is supervised by the Supervisor so that when it gets stopped or killed abruptly, Supervisor will start it automatically so that website is always keep running.

I am new to Django so if you have any suggestion or interested to contribute then you are always welcome to contribute in this project to make this a better site..
